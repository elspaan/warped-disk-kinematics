# Warped disk kinematics
Repo for all scripts and other documents related to uncovering warps in disks with kinematics.

You'll find:
* Scripts that make synthetic cubes observation-like
* Discminer scripts: clip+downsampling (`prepare_data_<disk>.py`); fitting (`fit_mc_img_<disk-specs>.py`)
* ... and any other analysis and modelling scripts.

Clip, downsample, fitting scripts are organised in folders per disk.
